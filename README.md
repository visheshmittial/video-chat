Video Chat App
- A real-time video chat application using React, Node.js, Socket.IO, and WebRTC. Users can video call and chat directly via web browserâ€”no signup required!

  Features
- Real-time peer-to-peer video calls via WebRTC
- Live text chat (messaging) system
- Fast and responsive user interface
- Simple room join system (auto-join or link-share)

 Tech Stack
- Frontend: ReactJS (JavaScript, HTML, CSS)
- Backend: Node.js, Express.js, Socket.IO
- Peer Connection: WebRTC
- Other Tools: npm, React Scripts, STUN Servers

Installation & Running
1. Clone the repository
text= 
git clone https://github.com/visheshmittial/video-chat.git
cd video-chat

2. Install backend dependencies
text=
cd backend
npm install
npm start

3. Install frontend dependencies
text=
cd ../frontend
npm install
npm start

Open http://localhost:3000 in your browser.
